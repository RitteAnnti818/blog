This is my first post. The blog is built with [Jekyll](https://jekyllrb.com/)
and hosted on GitHub Pages at `RitteAnnti818.github.io/blog`.

## Writing a new post

To add a post, create a Markdown file in the `_posts/` folder named
`YYYY-MM-DD-title.md`, with this front matter at the top:

```markdown
---
layout: post
title: "Your title here"
date: 2026-06-29
---

Your content in **Markdown**.
```

Push it to the `main` branch and GitHub Pages rebuilds the site automatically.
